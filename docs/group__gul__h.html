General Utility Library for C++17 1.2.3
gul17/gul.h

All-in-one include file for GUL17.

This file includes almost all other headers provided by the library. Two specific files are excluded, however:

  • gul17/catch.h: The header file for the Catch2 unit test framework
  • gul17/date.h: Howard Hinnant's header for calendar date functionality

While the unit test framework is simply not useful for inclusion into regular code, date.h is excluded chiefly because of its negative impact on compile times. Just include it directly where needed.
